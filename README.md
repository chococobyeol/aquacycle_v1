# AquaCycle

Windows와 macOS를 대상으로 개발 중인 2D 수중 생태계 시뮬레이션입니다. 현재 구현 범위에는 빛·조류·첫 소비자·미생물 물질 순환과 낮·밤을 다루는 도전 과제 6개, 자유 실험실, 구조물 물리 배치, 공간 수온·물 흐름·수질 측정, 표면 단위 조류 성장, 나사말, 체리새우의 섭식·번식·사망과 분해균·질산화균 필름이 포함됩니다.

현재 실행본은 초기 플레이어블에서 확인한 문제를 다시 설계한 배포 목표 구현본입니다. 한 번 클릭해 집고 다시 클릭해 놓는 배치 방식, 실제 스프라이트 비율과 일치하는 충돌 형상, 구조물 전면의 미세 조류·균 필름, 숫자형 탐침과 위치별 관찰 오버레이를 구현했습니다. 미션 1~4에서는 수질 효과를 꺼 기존 학습 범위를 유지하고, 미션 5·6과 실험실에서는 `36×20` 공간 열·부력 흐름과 같은 물 유량을 공유하는 유기물·독성 노폐물·영양염·용존산소·무기탄소 순환을 활성화합니다. 미션 6에서는 전등 없는 넓은 자연광이 낮·밤에 따라 변하고, 생산자의 총광합성과 호흡을 별도 물질 흐름으로 계산합니다. 실험실에서는 자연광과 천장 전등을 동시에 사용하되 두 광원이 같은 차폐·수심 감쇠·산란·반사 계산을 거쳐 합산됩니다. 생산자·새우·균·사체·물의 질소와 탄소는 하나의 보존 장부로 연결됩니다. 세부 기준은 [`docs/02_IMPLEMENTATION_SPEC.md`](./docs/02_IMPLEMENTATION_SPEC.md)에 정리되어 있습니다.

## 실행

```bash
npm install
npm start
```

## 검증과 패키징

```bash
npm run typecheck
npm test
npm run package
```

배포 전에는 장시간 생태 검사가 서로 CPU를 빼앗아 거짓 시간초과를 내지 않도록
전체 회귀를 한 작업자에서 실행하고, 낮·밤 50주기까지 이어서 확인합니다.

```bash
npm run verify:release
```

현재 검증 계약과 실행 기록은
[`docs/08_RELEASE_VERIFICATION.md`](./docs/08_RELEASE_VERIFICATION.md)에 남깁니다.

macOS 패키지는 `out/AquaCycle-darwin-*`에 생성됩니다. Windows에서는 같은 프로젝트에서 Squirrel 패키지를 만들도록 Electron Forge가 설정되어 있습니다.

## 구현 구조

- Electron: 데스크톱 창과 배포
- React: 메뉴, 미션, 실험실 UI
- PixiJS: 수조, 조류, 광량 필드 렌더링
- Matter.js: 돌의 중력, 충돌, 회전과 안착
- Web Worker: 물리·광량·성장 시뮬레이션
- 고해상도 두들 스프라이트 + 절차적 조류 표현

문서의 역할과 읽는 순서는 [`docs/00_DOCUMENT_INDEX.md`](./docs/00_DOCUMENT_INDEX.md)에서 한 번에 확인할 수 있습니다.

포함된 글꼴의 출처와 이용 안내는 [`THIRD_PARTY_NOTICES.md`](./THIRD_PARTY_NOTICES.md)를 참고하세요.
