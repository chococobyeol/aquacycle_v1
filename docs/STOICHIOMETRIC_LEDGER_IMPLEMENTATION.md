## 1. 수정 목적

기존 모델은 탄소와 질소 총량은 보존했지만 산소 비용을 반응별 경험계수로 계산했다.

- 생산자는 고정 탄소에 `1.12`를 곱해 산소를 생산했다.
- 새우 호흡은 탄소에 별도 계수 `0.86`을 사용했다.
- 분해균은 균체로 보존한 몫까지 포함한 전체 섭취량에 산소를 부과했다.
- 질산화는 산소를 소비했지만 질산염이 다시 생산자 생체 질소로 환원되는 반대 장부가 없었다.

따라서 같은 탄소·질소가 어떤 경로를 거치는지에 따라 한 순환 뒤 산소가 달라질 수 있었다. 광합성 계수 하나를 올리는 방식으로는 모든 경로를 동시에 닫을 수 없었다.

이번 수정의 목표는 다음과 같다.

1. 반응 속도와 물질 수지를 분리한다.
2. 먹기·배설·사망 같은 이동에는 산소를 부과하지 않는다.
3. 실제 산화·환원이 일어난 양에만 산소 등가량을 부과한다.
4. 탄소·질소뿐 아니라 산소 등가 불변량을 자동 검사한다.
5. 장기 낮·밤 실행이 임의 보정 없이 제한된 주기 궤도에 머무는지 확인한다.

## 2. 상태와 공통 환산계수

게임 생체량 `B`는 고정 조성을 사용한다.

```text
생체 질소 = 0.08 × B
생체 탄소 = 0.32 × B
```

게임 산소 단위와 유기탄소 단위의 환산척도는:

```text
qC = 1.12 O₂-equivalent / organic-C
```

질산화 질소의 환산척도는 독립적으로 튜닝하지 않고 실제 질량비에서 유도한다.

```text
탄소 산화: 32/12 g O₂/g C
암모니아성 질소 산화: 64/14 g O₂/g N

qN = qC × (64/14) ÷ (32/12)
   = 1.92 O₂-equivalent / nitrate-N
```

`qC`의 절대 크기는 게임 표시 단위의 스케일이지만, `qC:qN` 비율과 각 반응의 양방향 적용은 화학량론으로 고정한다.

## 3. 보존식

반응은 다음 형식으로 본다.

```text
dx/dt = S · v(x)
```

- `x`: 각 물질 저장고의 상태 벡터
- `S`: 반응별 화학량론 열
- `v(x)`: 광량·수온·기질·산소에 따른 반응속도

탄소와 질소 이외에 다음 산소 등가 장부를 둔다.

```text
E = O₂total - qC × Corganic + qN × Nnitrate
```

```text
O₂total = 용존산소 + 공기층 산소
Corganic = 모든 생산자 + 새우 + 균 + 사체 + 고형/용존 유기물의 탄소
Nnitrate = 영양염 저장고의 질산염성 질소
```

닫힌 내부 반응과 물–공기층 재분배는 `E`를 바꾸지 않아야 한다.

## 4. 반응별 구현

### 4.1 생산자 성장

```text
무기탄소 + 암모니아/질산염 → 생산자 생체량 + 산소 등가량

O₂ 생산 = qC × Cfixed + qN × Nnitrate-assimilated
```

암모니아성 질소는 유기 질소와 같은 환원 상태로 압축했으므로 추가 질소 항이 없다. 질산염은 유기 질소로 환원될 때 필요한 전자 흐름을 `qN` 산소 등가량으로 기록한다.

### 4.2 생산자·새우 호흡

```text
유기탄소 + 산소 → 무기탄소

O₂ 소비 = qC × Cmineralized
```

산소가 부족하면 수행 가능한 양만 생체·저장 질량에서 빠진다. 수행하지 못한 요청량을 먼저 삭제하지 않는다.

새우가 먹은 질량은 동화·배설·즉시 호흡 몫으로 나뉜다. 동화와 배설은 유기물 저장고 사이 이동이므로 산소 비용이 없다. 즉시 호흡 몫 중 산소가 부족해 무기화되지 못한 양은 고형 찌꺼기로 남는다.

### 4.3 분해균

```text
U = 처리한 유기 생체량
G = 새 균체로 보존한 생체량
M = U - G = 실제 무기화량

O₂ 소비 = qC × biomassCarbon × M
```

균체 `G`는 유기탄소 상태로 남는다. 나중에 죽어 찌꺼기가 되어도 아직 산화된 것이 아니므로 그 시점에도 산소를 부과하지 않는다.

### 4.4 질산화군집

```text
Nprocessed = 처리한 암모니아성 질소
Nretained = 새 균체 질소
Nnitrate = Nprocessed - Nretained
Cfixed = 새 균체 탄소

O₂ 소비 = qN × Nnitrate - qC × Cfixed
```

암모니아 중 균체로 바로 동화된 질소에는 질산화 비용이 없다. 암모니아 산화에서 얻은 환원력 중 균체 탄소 고정에 저장된 몫도 차감한다.

## 5. 한 스텝의 계산 방법

균 반응은 호출 배열 순서에 따라 결과가 달라지지 않도록 두 단계로 처리한다.

1. 유기물·암모니아·질산염·산소·무기탄소의 반응 전 배열을 복사한다.
2. 모든 기능군의 Monod 활동도와 요청 처리량은 이 공통 사전 상태에서 계산한다.
3. 기질·산소·무기탄소의 **인출 예산**을 별도로 만든다.
4. 암모니아·질산염·무기탄소의 **생성물 수용 예산**을 별도로 만든다.
5. 같은 스텝에서 생성한 물질을 다른 기능군의 사전 먹이로 사용하지 않는다.
6. 산소·기질·생성물 공간 중 가장 먼저 닿는 제한에 맞춰 반응량을 줄인다.
7. 제한된 반응 열 전체를 실제 저장고에 적용한다.

표면 배열을 정방향·역방향으로 전달해도 같은 결과가 나오는 시험을 추가했다.

농도장은 `Float64Array`를 사용한다. 32비트 저장에서는 매 스텝의 반응과 보존형 수송 자체는 대칭이어도 50주기 후 약 `0.053%`의 산소 등가 반올림 오차가 누적됐다. 64비트 전환 후 같은 실행의 오차는 약 `1.49×10⁻¹⁴`로 줄었다.

호흡 생성물의 저장 공간도 실제 반응 전에 확인한다. 다만 생물 한 마리가 호흡할 때마다 `36×20` 수조 전체를 합산하면 장기 실행이 크게 느려지므로, 먼저 반응점 주변 stencil의 공간을 검사하고 그곳이 실제로 포화된 예외에만 수조 전체 overflow 공간을 계산한다. 이 변경 뒤 미션 5의 7,200초 검증은 같은 개발기에서 약 90초 초과 상태에서 58초대로 줄었으며 반응량과 보존식은 바뀌지 않았다.

### 6.3 새우 개체군과 화학 평형의 구분

새우 4마리처럼 작은 유한 개체군은 화학 장부가 닫혀도 멸종할 수 있다. 번식 파동 뒤 먹이 부족, 같은 세대의 노화, 국소 야간 저산소, 우연한 성비가 원인이 될 수 있다. 따라서 다음 두 질문을 분리한다.

- **화학 구조가 새는가?** C·N·E 장부와 같은 위상의 주기별 수질로 판정한다.
- **이 배치의 유한 개체군이 살아남는가?** 출생·성숙·사망 원인과 실제 개체 수로 판정한다.

과밀이나 생산자 부족으로 새우가 멸종하는 것은 허용하지만, 그 과정에서 이유 없이 산소가 계속 사라지는 것은 허용하지 않는다.

## 7. 남은 단순화

- 게임 산소와 물질 값은 실제 mg/L·mol 단위가 아니다.
- 인·황·알칼리도·pH·탈질·메탄 생성은 아직 상태변수에 없다.
- 질산화는 암모니아와 아질산 산화를 하나의 통합 기능군으로 압축한다.
- 질산염 동화의 `qN`은 생략한 광합성 전자전달을 닫는 산소 등가 항이다.
- 실제 종별 C:N 조성 대신 하나의 감사용 고정 조성을 사용한다.
- 화학량론은 총량 드리프트를 제거하지만 특정 생물 구성의 안정성을 자동으로 보장하지 않는다.
